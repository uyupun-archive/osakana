# ダッシュボード

- ダッシュボードはViteとpreact-tsで構築されており、主にUIを提供する

## 環境構築

```bash
$ cd dashboard

$ cp .env.example .env

$ npm install

$ npm run dev
```

## リンター・フォーマッター

```bash
# ESLintの実行
$ npm run lint:eslint

# Prettierの実行
$ npm run lint:prettier

# ESLintの実行と問題の自動修正
$ npm run fix:eslint

# Prettierの実行と問題の自動修正
$ npm run fix:prettier
```
